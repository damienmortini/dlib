# `<element-zoomable>`

## Installation

```
npm install @damienmortini/element-zoomable
```

## Usage
```
<script type="module">

  import MyElement from '@damienmortini/element-zoomable';

  window.customElements.define(`my-element-name`, MyElement);

</script>

<my-element-name></my-element-name>
```
